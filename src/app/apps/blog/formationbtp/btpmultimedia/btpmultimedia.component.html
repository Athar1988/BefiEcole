Formation multimedia

Grace à cette formation, le technicien en multimédia sera capable d’assister à :

  • L’Analyse des besoins de l’entreprise ou bien du client en matière de communication
  • La conception (scénario, charte graphique, architecture technique)
  • Le développement, la production des différents supports de communication (Cdrom, site web, vidéo, films, émission de télévision..).

Les secteurs de recrutement sont:

  • Le département informatique de toute entreprise
  • Le département de communication de toute entreprise
  • Une boite de communication/ publicité
  • Le domaine d’E-Commerce
Cette formation développe chez le technicien en multimédia :
  • une passion pour les nouvelles technologies
  • la curiosité et la réactivité
  • Le sens de l’écoute afin de répondre aux attentes des partenaires
  • Une bonne maîtrise des langues (français, arabe et anglais)

Les modules de la formation :

  • Le système d’exploitation Windows et Linux
  • Architecture système informatique
  • Application bureautique (MS Word, MS PowerPoint)
  • Cryptographie & Sécurité des données informatiques
  • Français
  • Anglais
  • Arabe
  • HTML
  • Internet
  • Gestion d’entreprise
  • Gestionnaire de base des donnés (MS ACCESS)
  • Système d’exploitation réseau
  • Législation sécurité et hygiène
  • Observer la réalité du milieu professionnel
  • Langage scripts
  • Logiciel Auteurs
  • Logiciel de création de site web (Microsoft, Frontpage, Macro Média, Dreaweaver)
  • Commerce électronique
  • Interface homme-machine
  • Traitement d’image photos
  • Traitement son
  • Traitement vidéo (adobe première)
  • Animation 2D
  • Technologie de la communication
  • Expression plastique
  • S’intégrer au milieu professionnel
